Toggle is a binary on/off control. Use it for settings that take immediate effect
— enabling a feature, switching a mode, or opting in/out of a preference. For
choices that only commit on form submit, prefer a `Checkbox`.

          <li>When the default value matters — a Toggle clearly communicates the current state at a glance.</li>

        <h3>When <em>not</em> to use a Toggle</h3>

        <h2>Color is not a meaning</h2>

        <p>The on state uses brand color and the off state is neutral, but color alone is unreliable for users with color-vision differences. The toggle&rsquo;s position (thumb left vs right) and the screen-reader announcement carry the meaning.</p>

        <h2>Target size</h2>

        <p>The toggle&rsquo;s clickable area extends to the entire row including the label, comfortably exceeding the WCAG 2.5.5 minimum target size. Don&rsquo;t shrink the toggle with custom CSS.</p>
