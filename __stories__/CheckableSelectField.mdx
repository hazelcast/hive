CheckableSelectField is a multi-select that opens a dedicated panel with a
search input and explicit checkboxes for each option. Use it when users routinely
pick many values, want to filter as they go, and need the dropdown to stay open
across selections. For inline-chip multi-select, prefer `MultiSelectField`.
