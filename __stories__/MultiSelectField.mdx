<Title />

MultiSelectField lets users pick **multiple** values from a fixed list. Selected
values render as removable chips inside the control. Use it when the order of
selection doesn&rsquo;t matter and you need a compact summary inline. For
explicit checkboxes that stay open while picking, use `CheckableSelectField`.
