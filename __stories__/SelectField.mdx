<Title />

SelectField lets users pick **one** value from a fixed list of options. Use it
when the list is too long for radio buttons but short enough that a free-text
field would be wrong. For multiple selection use `MultiSelectField` or
`CheckableSelectField`. For very long lists where the user is likely to type,
prefer `AutocompleteField`.
