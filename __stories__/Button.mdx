        <p>A Button is composed of:</p>

        <ul>
          <li>Use <strong>contained + primary</strong> for the highest-priority action. One per page or grouping.</li>

          <li>Use <strong>color="warning"</strong> sparingly for actions the user should review before committing.</li>

          <li>Lead with a verb (<code>Create project</code>, not <code>Project creation</code>).</li>
          <li>Avoid generic labels like <code>OK</code>, <code>Submit</code>, <code>Click here</code>. Be specific about what happens.</li>

        <h2>Accessible name</h2>

The <strong>small</strong> variant can fall below that threshold with very narrow content (e.g. a single character). When using <code>size="small"</code>, ensure the label is wide enough, or pad it manually.</p>
